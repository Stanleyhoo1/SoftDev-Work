Why use Foundation?

The Foundation framework has extremely useful features. Some include scalable content, easily customizable, and has a lot of html templates to start with, making it great for beginners. It is also flexible and runs entirely on browser, making it easy for a Design specialist to get into it. It is made up of a 16-column grid, offering finer control and precision over the design. Foundation also works really well with SASS, unlike some other front-end frameworks.

Foundation is built with HTML, CSS, and JavaScript, core components for web design. It also has been used by large websites like Wordpress, Drupal, .Net, and many other companies. Foundation is also the framework professionally supported by an organization, namely, ZURB which also uses the framework every day for clients who have small business up to large companies. It has gone through five major iterations and many point releases to ensure its efficiency.

How to Guide

1. Include Foundation CSS and JavaScript

To use Foundation, add the following links to your <head> section:

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites/dist/css/foundation.min.css">
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/foundation-sites/dist/js/foundation.min.js"></script>

2. Initialize Foundation

Add the following script to initialize Foundation in your webpage:

    <script>
      $(document).foundation();
    </script>

3. Use Foundation Components

Here are examples of how to use Foundation's grid system, buttons, and forms:

Grid System

    <div class="grid-container">
      <div class="grid-x grid-margin-x">
        <div class="cell small-6">Column 1</div>
        <div class="cell small-6">Column 2</div>
      </div>
    </div>

Buttons

    <a href="#" class="button">Default Button</a>
    <a href="#" class="button alert">Alert Button</a>
    <a href="#" class="button success">Success Button</a>

Form

    <form>
      <label>
        Name:
        <input type="text" placeholder="Enter your name">
      </label>
      <label>
        Email:
        <input type="email" placeholder="Enter your email">
      </label>
      <button type="submit" class="button">Submit</button>
    </form>

4. Customize Foundation (Optional)

To customize Foundation, download its files from the official site and modify the SASS files or use the customizer tool.
